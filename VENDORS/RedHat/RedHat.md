CONTAINER REGISTRIES


registry.access.redhat.com requires no authentication, therefore does
not support redhat login nor registry token
registry.redhat.io requires authentication, supports redhat login and
registry token
registry.connect.redhat.com requires authentication, supports redhat
login and registry token, holds [Red Hat Partner
Connect](https://connect.redhat.com/) program images

NOTE: To retrieve content from an authenticated registry, you will need
to log into the registry using either your
Customer Portal credentials
OR
Red Hat Developer credentials
OR
Registry Service Account credentials -
https://access.redhat.com/terms-based-registry/
